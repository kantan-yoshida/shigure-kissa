# 時雨喫茶（しぐれきっさ）

雨の日だけ入ることができる、夜の横丁のドット絵喫茶店です。

現在地の天気を [Open-Meteo](https://open-meteo.com/) で判定し、**雨・霧雨・にわか雨・雷雨のときだけ**入店できます。
店内では、コーヒーを飲むウサギ・本を読むクマ・店主のネコ（マスター）をクリックすると、吹き出しで話します。

## 遊び方

1. 「空を見上げる」で現在地の天気を判定（位置情報の許可が必要）
2. 雨ならドアが開き、店内へ
3. キャラをクリックすると吹き出しが出ます
   - 🐰 ウサギ … [Advice Slip API](https://api.adviceslip.com/) の助言（日本語訳）
   - 🐻 クマ … [dummyjson](https://dummyjson.com/) の名言（日本語訳）
   - 🐱 マスター … 時雨喫茶の紹介（手書き）

> 晴れの日は「そっとのぞく（デモ）」から店内を見られます。

## 技術メモ

- ビルド不要の静的サイト（HTML / CSS / Canvas + JavaScript）
- ドット絵はすべて Canvas に矩形でプログラム描画
- 翻訳は [MyMemory](https://mymemory.translated.net/)（無料・キー不要・匿名）
- **HTTPS で配信が必要**（位置情報・fetch のため）。`file://` の直開きでは天気判定が動きません

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | ページ構造・入店ゲート |
| `style.css`  | スタイル・雨・吹き出し |
| `scene.js`   | ドット絵シーン（外観／店内）の描画 |
| `weather.js` | 天気による入店判定 |
| `talk.js`    | キャラのクリック→吹き出し |

## ライセンス / クレジット

外部API: Open-Meteo, Advice Slip, dummyjson, MyMemory。
